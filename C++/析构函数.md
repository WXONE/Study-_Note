# 析构函数

- 完成对象被删除前的一些清理工作。
- 在对象的生存期结束的时刻系统自动调用它，然后再释放此对象所属的空间。
- 如果程序中未声明析构函数，编译器将自动产生一个默认的析构函数，其函数体为空。
- 构造函数和析构函数举例



在类中，必须要有构造函数和析构函数，析构函数的语法跟构造函数有点像，它也是要用类名但是在类名前面呢，有一个小的波折号

### 析构函数例：

```c++
#include 
using namespace std;
class Point {     
public:
  Point(int xx,int yy);
  ~Point();
  //...其他函数原型
private:
  int x, y;
};
```
